# BuildThemUp
Build Them Up is a shooter tower defense game that merges strategic defense and resource management, challenging players to protect their base by upgrading turrets to counter increasing waves of enemies. 
> [!NOTE]
> The Game was released on [Itch.io](https://theplaceholdergames.itch.io/buildthemup). _Images and details are ripped from the itch.io page._

Build Them Up was created and produced by a group of Computer science and Game design students at Swinburne University. 
> [!IMPORTANT]
> Credits: [ThePlaceholderGames](https://theplaceholdergames.itch.io/) Itch.io Page.

## Controls
> [!TIP]
> **WASD** - Move
> 
> **Left Click**- Shoot / Place Turret
>
> **Right Click** - Aim down sights ( If weapon allows it )
>
> **Space** - Jump
>
> **1 to 5 OR Mouse Wheel** - Swap Weapon
>
> **E** - Interact / Open OR Close Menu
>
> **Q** - Switch Turret ( Whilst Holding TurretSpawner Tool
>
> **ESC** - Pause Menu

## Install instructions
### Windows
1. Download the game .zip file.
2. Extract the contents of the game’s .zip file to your desired directory.
3. Navigate to the extracted game’s folder.
4. Execute ‘BuildThemUp.exe’.
> [!NOTE]
> Congratulations! You have successfully installed Build Them Up for Windows.
